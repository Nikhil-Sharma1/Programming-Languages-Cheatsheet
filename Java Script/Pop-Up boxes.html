POP-UP BOXES

Pop-Up boxes are used to display the message or notification to the user without using printing satement.

Printing Statement

DOM element is used. i.e. document.write("text")

Types of Pop-Ups

  • Alert Box
  • Prompt Box
  • Confirm Box

  1. Alert Box

    2. It is used to put some warning to user. Example-:
    • Password is incorrect
    • username invalid
    • Battery low

  2. Prompt Box

    3. It is used to take input from user. Ther are two ways to get input-:
    1. Using prompt up box
    2. Using forms

  3. Confirm Box

    4. Whenever an action OR clicked by the user , the system asks for information. Example-:
    • Are you sure to end test
    • Are you sure to close allthe tabs
    It is used when the action is sensitive where no change can be performed after the action is performed.
