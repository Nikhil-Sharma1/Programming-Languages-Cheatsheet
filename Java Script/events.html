EVENTS

Javascript's interaction with HTML is handled through events thet occur when the user or
the browser manupulates a page.
When the page loads, it is called an event. When user clicks a button, that click too is an event.
Other examples incudes events like pressing any key, closing a window, resizing a window etc.

Types of events

  1. onclick Event

    2. This event occurs when a user a clicks the left button of his mouse. You can put your validation, warning etc., against this event type.

    Example

  2. onsubmit Event

    3. This event occurs when you try to submit a form. You can put your form validation against this event type.

    Example

  3. onmouseover Event

    4. This event triggers when you bring your mouse over any element.

    Example

  4. onmouseout Event

    5. This event occurs when you move your mouse out from that element.

    Example

  5. onkeypress Event

    6. This event occurs when the user pressed a key.

    Example

  6. onload Event

    7. This event occurs when an object has been loaded.

    Example

    Input Events

    Events description
    onblur When a user leaves an input field
    onchange When a user changes the content of an input field
    onchange When a user selects a dropdown value
    onfocus When an input field gets focus
    onselect When input text is selected
    onsubmit When a user clicks the submit Button
    onreset When a user clicks the reset button
    onkeydown When a user is pressing/holding down a key
    onkeypress When a user is pressing/holding down a key
    onkeyup When a user releases a key
