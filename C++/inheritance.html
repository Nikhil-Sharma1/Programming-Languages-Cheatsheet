INHERITANCE

Inheritance

means reusability. When you want to use data mambers and data functions of other class without friend function, inheritance is used.

Syntax-:

class2:"mode of inheritance"class1

Example-:

class A
{
int roll;
get()
{
cout<<"a";
}
put()
{
cout<<"b";
}
};
class B:A//members of A an B also
{
int marks;
getdata();
putdata();
}

Toatal members of B=own members+ inherite members from A 
marks+getdata()+putdata()+roll+get()+put()

Types of inheritance

  • Single Inheritance
  • Multiple Inheritance
  • Multilevel Inheritance
  • Hierarchical Inheritance
  • Hybrid Inheritance

  1. Single Inheritance

    2. In single inheritance, ther is only 1 base class and 1 derived class.

    Syntax-:

    derived class:"mode of inheritance"base class

  2. Multiple Inheritance

    3. In multiple inheritance, there are 2 base classes and 1 derived class.

    Syntax-:

    derived class:"mode of inheritance" Base1, "mode of inheritance"Base class 2

  3. Multilevel Inheritance

    4. In it, there is multiple level of inheritance. It requires minimum three classes. class A(members of A)-->class B(members of A+B)-->class C(members of A+B+C)

  4. Hierarchical Inheritance

    5. In hierarchical inheritance, there is 1 base class and multiple derived class.

    Syntax-:

    class A class B:public A class C:public A

  5. Hybrid Inheritance

    6. In hybrid inheritance, there is the mixture of any two types of inheritance.
