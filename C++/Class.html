CLASS

Everything in C++ is associated with classes and objects, along with its attributes and methods.For example: in real life, a car is an object. The car has attributes, such as weight and color, ANd methods, such as drive and brake.

Class

Class is a user defined datatype that we can use in our program, and it works as an object constructor, or a "blueprint"for creating objects. Class in C++ is same as structure in C, but we can include member functions and data memebers in it which can not be done in structures. Attribute and methods are basically variables and functions that belong to the class. These are often referred to as "class members".

Create a Class

To create a class , "class" keyword is used.

Example

class MyClass//

create a class of my class

{
public:
//access specifier

int myNum;
//data member

void function();
//data function
};

Create an object 

In C++, an object is created from a class. We have already created the class named Myclass, so now we can use this to create objects.
To create an object of MyClass, specify the class name, followed by the object name.
To access the class's data members and data functions, use dot syntax(.) on the object.
