STRUCTURES

STRUCTURE

are the set of homgeneous elements(all elements are of same type). It is declare by using struct keyword. The struct keyword can be using inside or outside the main function.

Example

struct student { char name[10]; int roll; int age; float marks; }; main() {struct student S={"hello",2,20,40.4}
printf("enter details-:Name,roll,age,marks");
scanf("%s%d%d%f",S.name,S.roll,&S.age,&S.marks);
printf("details are-:\n");
printf("Name=%s",S.name);
printf("Roll no.=%d",S.roll);
printf("Age=%d",S.age);
printf("Marks=%f",S.marks);
