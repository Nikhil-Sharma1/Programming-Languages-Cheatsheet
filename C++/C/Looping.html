LOOPING

Looping

is the process of call the function again and again for a specific time.

Types Of Loop

  • For Loop
  • While Loop
  • Do While Loop

  1. For Loop

    2. This loop is used when you already know how many times function is called.
    Syntax-:
    for(initialisation;condition;updatation)
    {
    statements;
    }

    Example

    int i;
    for(i=1;i<=20;i++)
    {
    print("Hello");//it will print hello for 20 times
    }

  2. While Loop

    In this loop, the function is called again and again when condition is true.
    It stops when the specified condition becomes false.
    Syntax-:
    while(condition)
    {
    statements;
    }

    Example

    int i=0;
    while(i<20)
    {
    printf("hello");//it will print hello for 20 times
    }

  3. Do While Loop

    In this loop also, the function is called again and again when condition is true. It stops when the given condition becomes false.
    But in this loop, condition is given at bottom of the loop.
    Syntax-:
    do
    {
    statements;
    }while(condition);

    Example


    int i=0;
    do
    {
    print("Hello");//it will print hello for 20 times
    }while(i<20)
