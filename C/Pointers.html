POINTERS

pointers

are used to store the address of other variables.

Pointer Declaration

datatype *pointer name;

Example

int *p //can store address of only integer variable
int a=10;
p=&a;//store address

Dereferencing The Pointer

int *p;
int a=10;
printf("%d",*p);//output:10
*p=20;
printf("%d",*p);//output:20
printf("%d",a);//output:20

Pointer Flexibilty

int *p,i,j,k;
i=2;
j=4;
k=6;
p=&i;
printf("%d",*p);//output:2
p=&j;
printf("%d",*p);//output:4
p=&k;
printf("%d",*p);//output:6

Types Of Pointers

  • Null Pointer
  • Wild/Uninitialised Pointer
  • Dangling Pointer
  • Void Pointer

    1. Null Pointer

      2. int *p; P=null;

    2. Wild/Uninitialised pointer

      3. int *p; int a; p=&a;//If this line is missing, then it is wild pointer. It does not provide any address.

    3. Dangling Pointer

      4. A pointer is pointing to a memory address of a variable and the variable is not existing, but the pointer is still pointing to the same address, this pointer is known as dangling pointer.
      main()
      {
      int *p;
      {
      int a=10;
      p=&a;
      }
      printf("%d",*p);//output:10, a is not accessible outside the block, but the pointer is still pointing towards a.
      }

    4. Void/Generic pointer

      5. In it, we can use any datatype, only there is some change in printf statement. It is also known as generic pointer.
      void *p;
      int a=1;
      float b=4.4;
      char c="x";
      p=&a;
      printf("%d",*(int*)p;
      p=&b;
      printf("%f",*(float*)p;
      p=&c;
      printf("%c",*(char*)p;
