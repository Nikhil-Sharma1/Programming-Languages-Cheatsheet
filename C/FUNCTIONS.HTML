FUNCTIONS

FUNCTIONS

are the block of statements used to perform specified task. There are two types of functions-:
  1. Library(pre-defined) functions
  2. User defined functions

Elements Of Funcions

  • Function declaration
  • Function call
  • Function definition
  1. Function Declaration
    2. Syntax-: return type function name (argument(datatype)/parameter);
  2. Function Call
    3. Syntax-: function name(argument(datatype);
  3. Function Definition
    4. Syntax-: return type function name(argument(datatype)/parameter) (here ';' sign is not used)

Full Syntax

function declaration;
main()
{
function call;
}
function definition
{
statements;
}
