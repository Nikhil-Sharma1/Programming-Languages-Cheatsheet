Q. What are the advantages of JavaScript?

Ans.

  • Less server interaction-:
    • You can validate user input before sending the page off to the server.
  • Immediate feedback to the visitors-:
    • They don't have to wait for a page reload to see if they have forgotten to enter something.
  • Increased interactivity-:
    • You can create interfaces thet react when the users hovers over them with a mouse or activates them via the keyboard.

Q. Is JavaScript case-sensitive?

Ans.

Yes, JavaScript is a case-sensitive language. This means that language keyboards, variables, function names
and any other identifiers must always be typed with a consistent capitalization of letters.
