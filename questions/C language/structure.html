Q. What are the various operations that can be performed on different data structures?


Ans.

Insertion-:

Add a new data item in the given collection of data items.

Deletion-:

Delete an existing data item from the given collection of data items.

Transversal-:

Access each data item exactly once so that it can be processed.

Searching-:

Find out the location of the data item if it exists in the given collection of data items.

Sorting-:

Arranging the data items in some order i.e. in ascending order or descending order in case of numerical data and in dictionary order in case of alphanumeric data.
