Q. What are C++ access specifiers?

Ans.

Access specifiers are used to define how the members(functions and variables) can be accessed ouside the class.
  • Private:
    • Members declared as private are accessible only within the same class and they cannot be accessed outside the class they are declared.
    Child classes are also not allowed to access private members of parent.
  • Public:
    • Members declared as public are accessible from anywhere.
  • Protected:
    • Only the class and its child classes can access proetected members.
