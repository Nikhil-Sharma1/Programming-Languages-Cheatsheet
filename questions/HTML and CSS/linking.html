Q. In how many ways can a CSS be integrated as a web page?

Ans.

CSS can be integreted in three ways-:
  • Inline-:
    • Style attribute can be used to have CSS applied HTML elements.
  • Internal-:
    • The HEAD element can have a style element within which the code can be placed.
  • External-:
    • CSS can be placed in an external file and linked via link element.
