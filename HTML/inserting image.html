INSERTING IMAGE

HTML

allows us to placing of images in a web page. HTML accepts two types of picture formats i.e .gif and .jpg.
We can copy images by using path of an image and paste on a web page.

Insert an Image

To insert an image on a web page, img tag is used ehich takes file name and format name. syntax-:img src="file pathway"

Types of pathways

  • Absolute Pathway
  • Relative Pathway
  1. Absolute Pathway
    2. Image is in the same folder where the HTML document is present. Syntax-:img src="filename.ext"
  2. Relative Pathway
    3. Image is in some other folder. Syntax-:img src="full pathname.ext"

Attributes of img tag-:

  • src

means source attribute. It specify the address of picture or graphic file.

  • Align

    • It allows to place the image to any direction(up,down,right or left).

  • Border

    • It specify border around image.

  • Height

    • It specify height of image in pixels.

  • Width

    • It specify width of image in pixels.

  • Alt

    • Means alternate. It indicates the text to be disply in case the browser unable to display image specified with img tag.
